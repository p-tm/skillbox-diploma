Бот по поиску отелей

Описание основных команд

Основные команды вводятся вручную или через меню (синяя кнопка "Menu" в левом нижнем углу

  • /start - начинать работу с ботом необходимо с этой команды; до ввода этой команды бот не может принимать команды от пользователя
  • /stop - останов выполнения команды
  • /lowprice - подобрать самые дешёвые отели
  • /highprice - подобрать самые дорогие отели
  • /bestdeal - сделать подборку отелей в заданном ценовом диапазоне и находящихся на заданном расстоянии до центра
  • /history - посмотреть историю поисков с результатами каждого поиска

При выполнении каждой из команд процесс выполнения занимает несколько шагов.

Для просмотра более подробной информации по каждой из команд выберите команду при помощи кнопки под сообщением

Команда "/start"

Выполните эту команду для начала работы с ботом

Команда "/stop"

Выполните эту команду, если Вы начали выполнение какого-либо запроса (например, /lowprice), но затем решили прервать запрос.

Команда "/lowprice"

Эта команда позволяет выполнить поиск самых дешёвых отелей. После нажатия этой команды необходимо ввести дополнительные данные:

  • выбор страны (выбор производится при помощи кнопок)
  • выбор города (выбор производится при помощи кнопок)
  • выбор количества отелей (ввод числа)
  • указание, нужны ли фото (кнопки "да/нет")
  • если нужны фото, то далее идёт выбор количества фото (ввод числа)
  • выбор даты заезда (с виртуальной клавиатуры)
  • выбор даты выезда (с виртуальной клавиатуры)
  • Вам будет показано резюме по выбранным Вами опциям, и после этого будет произведён запрос к удалённому сервру (это займёт некоторое время)
  • После получения ответа от сервера будет выведена информация о каждом из отелей последовательно, с добавлением фото к описанию каждого отеля (если такая опция была выбрана)

  • выбор даты выезда (с виртуальной клавиатуры)
  • Выбор минимальной стоимости (ввод числа)
  • Выбор максимальной стоимости (ввод числа)
  • Выбор минимального расстояния от центра (ввод числа)
  • Выбор максимального расстояния от центра (ввод числа)
  • Вам будет показано резюме по выбранным Вами опциям, и после этого будет произведён запрос к удалённому сервру (это займёт некоторое время)

Команда "/history"

Выводит историю действий пользователя и историю поиска, а также выводит результаты поиска по каждому поисковому запросу
